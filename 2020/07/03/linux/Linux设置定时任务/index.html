  2. 添加定时任务：

    在打开的编辑器中，你可以添加定时任务的行。每一行的格式如下：

    • minute：分钟 (0 - 59)
    • hour：小时 (0 - 23)
    • day：日 (1 - 31)
    • month：月份 (1 - 12)
    • day_of_week：星期几 (0 - 7, 其中 0 和 7 都表示星期日)
    • command_to_run：要运行的命令
