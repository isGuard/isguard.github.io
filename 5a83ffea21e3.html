Windows

在 Windows 中的命令提示符（Command Prompt）和 PowerShell 中，分号 (;) 也可以用来分隔多个命令，使它们依次执行。与 && 不同的是，使用分号分隔的命令不会检查前一个命令是否成功。

在命令提示符中

在命令提示符中使用分号 (;) 作为命令分隔符不总是直观的做法，因为它可能不会像在 PowerShell 中那样一致地工作。通常建议使用 & 或 && 代替。
