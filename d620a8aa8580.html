    1. ALTER DATABASE 数据库名称 SET SINGLE_USER WITH ROLLBACK IMMEDIATE
  2. 关闭所有查询窗口,修改 Options 的 Collocation 属性为 Chinese_PRC_90_CI_AS

    1. ALTER DATABASE 数据库名称 SET MULTI_USER
